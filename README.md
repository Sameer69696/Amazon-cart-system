# Amazon-cart-system
This project is a multi-page Amazon website clone built using HTML, CSS, and Vanilla JavaScript.
It dynamically renders products by generating HTML inside JavaScript and injecting it into the DOM, instead of hardcoding product markup.
The application also includes cart management and order tracking, simulating the core flow of a real-world e-commerce platform.

# Project Overview

i. Multi-page application (Home, Cart, Orders, etc.)

ii. Product data managed in JavaScript objects

iii. Dynamic product rendering using template literals

iv. DOM manipulation for updating UI

v. Event-driven interactions (Add to Cart, Order tracking)

# Features

i. Amazon-style product listing page

ii. Dynamic product grid rendering

iii. Star rating and product details display

iv. Add to Cart functionality

v. Cart persistence across pages

vi. Order history and tracking page

vii. Clean, modular JavaScript architecture

viii. No frameworks (pure HTML, CSS, JavaScript)
